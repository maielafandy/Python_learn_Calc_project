# Python Calculator Project

A simple calculator project built using Python for learning purposes.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Error handling
* Loop system
* Exit option

## Technologies

* Python

## What I Learned

* Input and output
* Data types (`int`, `float`, `string`)
* Conditions (`if`, `elif`, `else`)
* Loops (`while`)
* Error handling using `try` and `except`

## How to Run

```bash
python Calculator.py
```
